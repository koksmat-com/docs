Niels discusses his journey from a traditional Microsoft infrastructure to adopting Kubernetes and OpenShift, emphasizing the use of GitHub Actions for managing configurations. He shares insights and experiences from various conferences and presentations on cloud-based tools. The talk covers a practical demonstration on how to set up GitHub Actions for internal team use, highlighting the automation and integration capabilities. Niels also touches on personal anecdotes, including challenges with autism and ADHD, and their impact on his work style. The presentation includes insights on Microsoft 365, difficulties with unit tests, and the importance of modular coding. The session concludes with an invitation for audience interaction and feedback, particularly on developing a GitHub operator for better orchestration and error handling.
